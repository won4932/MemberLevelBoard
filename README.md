# MemberLevelBoard

## 개요
멤버별로 권한을 부여해서 게시판에 접근할 수 있는 각 권한에 따라서 게시판에 접근 가능한 게시판

## Environment
- Language : PHP
- Server : Apache24 2.4.41
- Db : Mysql 15.1 Distrib 10.1.20-MariaDB
- Tool : VSCode
- Category : PC, Web

## 상세기능
글작성, 댓글 작성, 파일첨부, 로그인, 회원가입, 댓글 카운트, 조회수, 댓글/게시판 비밀번호, 게시판 별 권한 부여, 권한 별 포인트 부여, 게시판 페이징
